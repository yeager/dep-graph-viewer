# Dependency Graph Viewer

Visualize dependency graphs for .deb packages.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install dep-graph-viewer
```
